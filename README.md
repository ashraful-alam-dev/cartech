🏎️ CarTech | Premium Automotive Experience
A high-end, responsive automotive landing page built with React and Tailwind CSS. CarTech offers a seamless, luxury-focused interface for browsing elite vehicles, accessories, and professional automotive services.

✨ Features
Premium UI/UX: A sleek "Dark Mode" aesthetic using a custom Gray/Amber color palette and glassmorphism.

Dynamic Car Gallery: Filterable inventory with detailed modal views for every vehicle.

Interactive Shop: A dedicated product section for accessories with real-time search functionality.

Service Overview: High-impact service cards showcasing financing, maintenance, and concierge support.

Fully Responsive: Optimized for all devices, from mobile phones to ultra-wide monitors.

Modern Navigation: Fixed glassmorphism navbar with a smooth mobile menu.

🛠️ Tech Stack
Framework: React.js

Styling: Tailwind CSS

Icons: Lucide React (or SVG Pathing)

Build Tool: Vite

🚀 Getting Started
Prerequisites
Node.js (v14 or higher)

npm or yarn

Installation
Clone the repository:

Bash
git clone https://github.com/Ashraful-Alam-Dev/cartech.git
cd cartech
Install dependencies:

Bash
npm install
Run the development server:

Bash
npm run dev
Build for production:

Bash
npm run build

🎨 Design Identity
Primary Background: bg-gray-900

Accent Gradient: from-amber-400 to-orange-500

Borders: border-gray-800 (Subtle) & border-gray-400 (Nav/Footer Divider)

Typography: Bold, uppercase tracking for headers to maintain a luxury automotive feel.
